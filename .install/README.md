# inventory
need `stancerkit` item.

Example:

```
stancer_kit = { name = 'stancer_kit', label = 'Stancer Kit', weight = 100, type = 'item', image = 'stancerkit.png', unique = false, useable = true, shouldClose = true, description = 'Stancer Kit' },
```
